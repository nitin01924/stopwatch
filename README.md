# ⚙️ How It Works

```text
Start Button
      │
      ▼
setInterval()
      │
      ▼
Update Every Second
      │
      ▼
Display HH:MM:SS
      │
      ▼
Stop / Reset
```

---

# 🎮 Controls

| Button   | Action             |
| -------- | ------------------ |
| ▶️ Start | Starts the timer   |
| ⏸️ Stop  | Pauses the timer   |
| 🔄 Reset | Resets to 00:00:00 |
